# telegraph

Минималистичная система быстрой публикации статей на PHP и SQLite.

## Что умеет

- создать статью по заголовку и содержимому;
- получить короткую ссылку вида `/a3f91c7e02bb`;
- редактировать статью по авторскому cookie;
- загружать изображения;
- вставлять YouTube и Vimeo;
- показывать статью серверным HTML без зависимости от Editor.js на просмотре;
- управлять статьями через админку;
- отдавать `sitemap.xml`.

## Локальный запуск

```powershell
.\start-local.ps1
```

Открыть:

```text
http://127.0.0.1:8000/
```

Админка:

```text
http://127.0.0.1:8000/admin.php
```

При первом заходе в админку нужно создать логин и пароль.
